# User Stories

Prioritized by tier. P0 = MVP must-have, P1 = important for usability, P2 = nice-to-have.

## P0 -- Core MVP

### Authentication

- As a user, I want to register with email and password so that my data is private and persisted.
- As a user, I want to log in and stay authenticated so I do not need to re-enter credentials every session.

### Exercise Management

- As a user, I want to browse my exercises filtered by muscle group so I can quickly find what I need.
- As a user, I want to create exercises with a name, description, and category so I can build my personal library.
- As a user, I want to edit an exercise's details so I can fix mistakes or refine descriptions.
- As a user, I want to delete an exercise I no longer use so my library stays clean.

### Plan Builder

- As a user, I want to create a workout plan with a name and a list of exercises so I have a structured routine.
- As a user, I want to configure target sets, reps, and rest time per exercise in a plan so the plan matches my training goals.
- As a user, I want to edit a plan (add, remove, or reorder exercises) so I can evolve my routine over time.
- As a user, I want to delete a plan I no longer follow.

### Live Workout

- As a user, I want to start a workout from one of my plans so I get guided through each exercise.
- As a user, I want to log each set with weight and reps as I complete it so my workout is recorded in real time.
- As a user, I want to skip a set I cannot complete (injury, fatigue) so I can move on without blocking my workout.
- As a user, I want to complete a workout and see a summary so I know what I accomplished.
- As a user, I want to abandon a workout early and keep the sets I already logged so partial data is not lost.

## P1 -- Important for Usability

### Exercise Library

- As a user, I want to search exercises by name so I can find them without scrolling through categories.
- As a user, I want to see exercise details (description, images) so I know how to perform the movement correctly.

### Live Workout

- As a user, I want a rest timer that counts down between sets so I stay on pace.
- As a user, I want the app to recover my in-progress workout if the app crashes or restarts so I do not lose my session.
- As a user, I want the screen to stay awake during a workout so I do not have to keep unlocking my phone.
- As a user, I want haptic feedback when a set is logged or a timer ends so I get physical confirmation.

### Plan Builder

- As a user, I want to drag exercises to reorder them in a plan so I can arrange my routine intuitively.
- As a user, I want to add per-exercise notes in a plan (e.g., "pause at bottom") so I remember technique cues.

### History

- As a user, I want to see a list of my past workouts with date and plan name so I can review my training.
- As a user, I want to view the full details of a past workout (every set) so I can see exactly what I did.

## P2 -- Nice-to-Have

### Progress Tracking

- As a user, I want to see my personal records (max weight, max reps, max volume) per exercise so I can celebrate milestones.
- As a user, I want to see a chart of my exercise history over time so I can visualize progression.
- As a user, I want to see volume trends across workouts so I know if I am training enough.

### Exercise Images

- As a user, I want to upload images to an exercise so I have a visual reference for form.

### Polish

- As a user, I want empty states that guide me (e.g., "No exercises yet -- create your first") so I am never confused by a blank screen.
- As a user, I want loading indicators on all data fetches so I know the app is working.
- As a user, I want clear error messages when something goes wrong so I know how to fix it.
- As a user, I want a dark theme so the app is comfortable to use in a gym environment.
